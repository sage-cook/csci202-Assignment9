What I Have Yet to Complete (In Order of Importance):

  1. Ability to save/load circuit layouts
  2. Menu for saved circuits
  3. Named tunnels (and maybe colors/shapes to tell them apart)
  4. Simulation controls which allow you to set simulation speed, start/pause/stop, and designate inputs from tunnels
  5. Adjustable size of circuit grid
  6. Nested Circuits: this is a major undertaking and will be split up into pieces.
    1. Controls to specify whether circuit should be baked or nested.
    2. Baked Circuits: these work by first simulating the circuit with every possible set of simultaneous inputs and generating a truth table based on the outputs. This then produces a tile you can place in other circuits whose behavior is determined by that truth table. This type of circuit is best used for components that process signals, like logic gates, since it's less resource intensive and processes all inputs in a single step.
    3. Real Nested Circuits: these work by simulating their circuit alongside the containing circuit, and simply routing all incoming ants to the appropriate tunnels and all outgoing ants to the appropriate sides of the tile. These will probably be harder to implement despite being easier to understand, so I'm putting them second. This type of circuit is best used for components which need to remember their state or deal with timing, since these are things that baked circuits can't do.
    4. Some pre-built starter circuits. This game would be very frustrating if you were just tossed in with nothing but paths and ants, so the player should start with some basic components to play around with. They should not be able to see the inner workings though, as they will eventually be encouraged to design their own versions of these components.
    7. This is where I think I will be able to get to by the due date. Anything after this is just extra.
  7. Ability to save/load game progress (Autosave?)
  8. Circuit tests: see if your circuit produces a desired output
  9. Currency System: buy materials/components and sell circuits for profit
  10. Tutorial

The Game Thus Far:

Instructions: Draw paths by left-clicking and dragging with the draw tool. The ant tool places / rotates ants with left click. The tunnel tools place tunnels on the edge of the board with left-click. Right-click or shift-click with any tool erases instead. Click the "Simulate!" button to run one step of simulation.

You can find the current build of the game at https://greenbeansagainstveganism.github.io/AntMachineGame/
